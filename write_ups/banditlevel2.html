Write up

  1. Step one would be to locate the home directory. The home directory is usually signified with a ~$ and is usually the directory you start in when logging into a Linux box. So it appears we are in the home directory.
  2. Step two, Now we need to search the directory for a file called ‘spaces in this filename’. Using the ‘ls’ command we see a list of all the files and folders in the home directory. The ‘ls’ command shows that there is only 1 file and it is the file we are looking for. bandit2@bandit:~$ ls
    spaces in this filename
  3. Step three, we need to open or read the file. Files with spaces have to be treated a little differently when using them in a command prompt. We need to add “ ” quotation marks around the file name so that the command prompt will know all these characters, including the spaces, belong together. Lets use cat to read the file, cat “spaces in filename”. And here is the flag that we wanted to capture. This is the password to log into bandit level 3. bandit1@bandit:~$ cat "spaces in this filename"
    aBZ0W5EmUfAf7kHTQeOwd8bauFJ2lAiG     And here is the flag that we wanted to capture. This is the password to log into bandit level 3. (aBZ0W5EmUfAf7kHTQeOwd8bauFJ2lAiG)
  4. And don’t forget to use the login username of bandit2 for the next level.
